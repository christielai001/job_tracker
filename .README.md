# Job Application Tracker
A simple Django REST API for tracking job applications. This project allows authenticated users to create, view, update, and delete job applications.

## Table of Contents
- [Features](#features)
- [Technologies](#technologies)
- [Installation](#installation)
- [Environment Variables](#environment-variables)
- [Usage and API](#usage--and--api)
- [Project Structure](#project-structure)

## Features
- User authentication using JWT
- Create, read, update, and delete job applications
- Track the status of job applications (ex: applied, interview, offered, rejected)

## Technologies
- Python
- Django
- Django REST Framework
- Simple JWT
- SQLite

## Installation
- Clone repo
- Create virtual environment
- Install dependencies in requirements.txt
- Run migrations
- Start server

## Environment Variables
- Create a '.env' file in the project root:

        SECRET_KEY=(your secret key)

        DEBUG=True

        ALLOWED_HOSTS=127.0.0.1,localhost

## Usage and API
- Admin panel: htpp://127.0.0.1/admin/ 
- API base URL: http://127.0.0.1:8000/api/
- Endpoints can be tested using Postman

        GET  /api/job_applications/ - list of job applications

        POST /api/job_applications/ - create a job application

        PATCH /api/job_applications/{id}/ - edit/update job application

        DELETE /api/job_applications/{id}/ - delete job application

- Obtain a user token using POST request in Postman
